# Team Page

## Why CSS Grid?
- Used **CSS Grid** to create a clean and responsive card layout.
- Makes it easy to change the number of columns for different screen sizes.

## Responsive Breakpoints
- **> 1200px:** 4 columns (Desktop)
- **≤ 1200px:** 3 columns (Large tablets / small laptops)
- **≤ 900px:** 2 columns (Tablets)
- **≤ 600px:** 1 column (Mobile)
